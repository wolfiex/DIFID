A network consists of items and relationships

We begin by taking a number of reviewed papers and plotting them

These topics are extracted from a copus of papers.

Each paper has a relationship to one or more topics, however topics tend to cluster mostly around a single subject or idea.

Exploring the paper-topic relationships we can convert this itno a sociograph.

Here we have taken the papers which have a relationship value within the 95th percentile of the dataset and draw links between them and the topics for which they correspond.
If you interact with the graph we can see, that there are many links to the a groupings main topic, and a smaller amount to other topics.

Finally we can use graph clustering to get like topics based of the entire dataset.

This works by finding an underlying structure of the topic-paper relationship mapping rather than their location within the reduced space graph in the figure.
. We can now compare the groups discovered on the plotted graph.
